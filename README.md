# 개요
### todolist 만들기 </br>
  -> TODO  DOING  DONE 3가지 항목(type)으로 분류 </br>
  -> todo등록시 TODO항목에 저장 </br>
  -> 각 항목에 스케줄을 끝내면 다음 항목으로 이동 ( 데이터베이스 type 업데이트)
 
 # 기술
 1.백엔드 : 서블릿 및 JSP(EL 및 JSDL문법) </br>
 2.프론트엔드 : HTML CSS JavaSrcipt </br>
 3.데이터베이스 : MYSQL </br>
 4.서버 : Apache Tomcat </br>
